# Atrox Tax Calendar 2026

Професионален данъчен календар за счетоводители, изграден с Next.js 14.

## 🚀 Бърз старт

### Инсталация

```bash
# Клонирайте репото
cd atrox-tax-calendar

# Инсталирайте зависимостите
npm install

# Стартирайте development сървър
npm run dev

# Отворете http://localhost:3000
```

### Build за продукция

```bash
npm run build
# Генерира се папка 'dist' със статични файлове
```

## ✨ Функционалности

### 📅 Данъчен календар
- 147 системни задачи за 2026 г.
- ДДС декларации (месечни/тримесечни)
- Годишни данъци (АД, ЕООД, ЕТ)
- Осигуровки и заплати
- Справки по ДДС и Интрастат

### 🏢 Управление на фирми
- Добавяне на неограничен брой фирми
- Данни за ЕИК, МОЛ, ДДС режим
- Филтриране и търсене

### ✅ Чеклист
- Проследяване на изпълнението по фирми
- Маркиране на завършени задачи
- Филтриране по месец

### 🔔 Известия
- Автоматични напомняния за наближаващи срокове
- Приоритет: Критични (0-3 дни), Внимание (4-7 дни)

### 📊 Dashboard
- Обобщена статистика
- Предстоящи задачи
- Бързи връзки
